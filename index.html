HTML

  • The < head > element contains information about the webpage.
  • The < title > element defines the title for a webpage.
  • The < style > element defines the CSS styles associated with a website.
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata such as the charactrer set, description, keywords, author, and viewport.
  • The < body > element represents the visible content shown to the user.
  • An element refers to the entire element from opening tag to closing tag, wheras a "tag" refers only to what appars inside the angle brackets.
  • h1-h6 elements represent the leel of heading a text block represents.
  • The < p > element stands for a paragraph or block of text.
  • There are 3 different types of list options. < ul >,< ol >, and < li >.
  • Adding an image you will use the < img scr=" "/ > element. You can also use < img alt=" " / >.
  • The < br > element creates a line of empty space, or a line break, between 2 blocks of content.
  • Speaking of the hyperlink, the < a > element (which stands for anchor) creates links to the same webpage or other webpages. The href attribute points to the URL for the link.
  • The < header > will appear on the top of the page. Remember that order matters from < h1 > to < h6 >.
  • The < footer > element appears at the bottom of the page . Normally will contain the author, copyright, contact, sitemap, and navigation.
  • You can add < section > elements to the < body > of the HTML document. A section element is a semantic element that defines a section in a document that contains thematic content.
  • Inside the "section" element, we use class attribute and id attribute. HTML uses the "id" attruibute as a way to identify elements and select them.

CSS

  • CSS stands for Cascading Style Sheets.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • When a developer wants to style an element directly in the HTML file, they use the inline CSS.
  • It is also possible to embed an entire stylesheet directly in the HTML file using a < style > element in the < head > element. this internal style sheet contains CSS rules for the webpage in the head section of the HTML file. The rules only apply to that page, and you can configure CSS classes an IDs, which are used to identify elements, to style multiple elecment in the page code.
  • The most common way to style a webpage is by using external CSS style sheet.
  • When using an external style sheet, we link the file using a < link > element placedd in the HTML's < head >
  • NERD NOTE: Separation of concerns is a widely used term in web development and refers to separating code into different files based on use.
  • A selector and its declorations is called CSS RULE. An example is div { color: blue; text-align: justify;}
      "color:" and "text-align:" is called the Property.
      "blue;" and "justify;" is called the value.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Make sure when using the exteral style sheet that you < link > it to the HTML file.
      • < link rel="stylesheet" href="./assets/style.css">

    • The < link > element creates the connection to the external file.
    • The "rel" attribute specifies the relationship between the current document and the linked document resource.
    • The "href" attribute specifies the location (URL) of the external resource.

JavaScript

  • JavaScript is an object-oriented programming (OOP) language.
  • A variable is a named container that allows us to store data in our code.
  • The ".js" file extension indecates that this file is a JavaScript file.
  • In coding parlance, a variable is simply a named container that allows us to store data in our code.
  • To delcare a variable, use "var" keyword. then you give the variable a name. After you follow with an ="".
      • var shape ="circle";
  • In the example above, the "shape" is the name of the container, or variable , and the data it stores is "circle".
  • Note that each line of JavaScript code ends with a semicolon.
  • You can use the "console.log" method to output a message to the web console by adding an argunment.
  • Use the Chrome DevTools!
      • Right click while looking at the browser, select "Inspect" from the menu and click Consoleto open.
  • Make sure you connect the script.js file to your index.html file.
  • We use "console.log" to confirm that the < script > tag that we added to "index.html" properly connected the script.js with index/html.
  • Control flow is the order in which a computer executes code in a script.
  • Control flow can be interrupted, however. We can us conditional statement and loops to control the order in which code is executed, or if it is even executed at all.
  • An "if" statement is a way of introducing decision-making into our code. Just like its name sounds, when using the "if" statement, code will only execute if the statement meets a condition, or is true.
  • To create an "if" statement in JavaScript, we use the same logic and similar syntax, shown in the following code example:
      • if (condition) { // block of code to be executed if condition is truthy }
  • When the computer encounters an "if" statement in a JavaScript file, it will check to see if the condition that we provide is truthy. If it is, then it will do whatever action we ask it to do in between the curly brackets. If the condition isn’t true or has a value that isn’t truthy, then the computer will ignore whatever is in between the curly brackets and move on.
  • If you have more than one condition we can use the "else if"
  • We use "===" in each condition. The strict equality operator (===) checks to see if two values are equal, and returs a Boolean results "true" if the values are equal and "false" if the values are not equal.
  • A group of related data is called an array.
  • Use the (for) loop to perform a task on each piece of the data.
  • In JavaScript and many other programming languages, arrays are zero-indexed and sequential. This means that the first item in any array has an index of 0, not 1. The indices, or location identifiers, will always be whole numbers that increase by 1 each time.
  • A "for" loop contains the following 3 inportant statements
      • The first statement determines the starting point of our loop.
      The second statement is the condition.
      The final statement is what allows the array to iterate over each item.
  • Commenting out code makes it inactive without deleting it, just in case you need it later.
      • To comment out multiple lines of JavaScript code, put /* before and */ after the block of code.
  • A function is simply a set of instructions that tells the computer how to perform a certain task.
      • A function is different from a "for" loop or condidtional statement because functions do not automatically execute when the JavaScript file is run. Instead, you must call a function by name in order to execute it.
      Always follow a function with parentheses ().
  • To declare a function, we use the function keyword and then give our function a unique name followed by parentheses. Then we add curly brackets {} that contain the code for the task we want to perform.
